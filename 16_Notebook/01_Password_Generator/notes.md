<!-- ❓ Tell me about your project -->
I created a Password Generator application using HTML, CSS and JavaScript.
It allows users to select password length and character types like uppercase, lowercase, numbers and symbols.
It generates a random password and also checks its strength.

<!-- ❓ What technologies did you use? -->
I use HTML, CSS and JavaScript.

<!-- ❓ How does your project work? -->
First, I collect user inputs like password length and selected character types.
Then I combine all selected characters into one string.
After that, I use a loop and Math.random() to generate random characters.
Finally, I display the password and check its strength.
